#1.navigate to backend folder

cd backend

#2.run the commands: 

npm init -y
npm install express cors
npm install mongoose

#3.Run the backend

node server.js

- 💰 Donations stored 
- 📊 Live leaderboard based on total funds

- **Database**: MongoDB (with Compass)
